Introduction and Short History
Microcal Origin is an excellent program for Scientific and technical graphics. This program has a great importance for me because in 1991 it was my first windows target. My friend Dema ordered it via snail mail and we got a demo disk which contained a crippled version of this program version 1.0 (no saving + time limit). It was by reversing this program that I first came to know that some programs contained each and every routine (e.g save in this case) BUT disabled. So it is a matter of common sense to uncripple it.This program has the following features which i liked :

1. Best available numerical analysis and curve fitting toolbox.
2. 101 times better than Micro$oft EXCEL.
3. Plots Each and Every type of Scientific and technical graphs.

So its a time limit protection. Now lets move to the methods :
 

Method 1
We used this method in 1980s.
